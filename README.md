# TNT Substance Use Prevention Project

## Problem
Youth substance is increasing in King County, Washington. The County has tasked you with identifying stressors, lack of support and other factors and proposing a solution.

## Our Goal
Use data and machine learning to better understand and prevent risk.

## Tools 
- Python
- Goodle Colab
- Scikit-learn

## Team Project Prompt
Note: Each team will tackle the same problem but we aren't looking for the same solutions. Analyze the data and when it's time to talk solutions, get creative!

Problem: How might we use data and technology to reduce substance use risk in King County's youth communities?

## Expectations
This pilot program is only a 5-10 hour monthly commitment. Manage your time and ask for help when you need it. 

Collaborate with your teammates. This may require you to identify timeframes to do a bi-weekly call or sync over a Teams channel. 

Please provide feedback! We are doing something new and can't improve without your insights.
